# EduFlow AI

EduFlow AI is a production-grade multi-tenant learning analytics and workflow automation backend platform built using Go.

## Vision

Build a backend system demonstrating:

- Clean Architecture
- JWT Authentication
- RBAC
- PostgreSQL
- Redis
- Docker
- Microservices
- Event-driven architecture
- Kubernetes
- Observability
- AI-powered recommendations
- CI/CD
- Distributed systems concepts
